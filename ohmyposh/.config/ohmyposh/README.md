# Introduction

> Oh my posh is a good replacement for oh my zsh as it went into life support mode.

## Step 1: Install OhMyPosh

```bash
curl -s https://ohmyposh.dev/install.sh | bash -s
```

## Step 2: Stow this directory

```bash
cd ~/dotfiles
stow ohmyposh
```

## Setp 3: Source your zshrc

```bash
source ~/.zshrc
```
